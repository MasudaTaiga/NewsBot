# 【NewsBot】ITニュースクリップ 運用マニュアル

このGoogle Apps Script (GAS) は、Googleスプレッドシートに登録されたキーワードをもとにGoogleニュースから最新記事を収集し、重複度に基づいたスコアリングを行ってGoogle Chatへ通知するツールです。

## セットアップ手順

### 1. スプレッドシートの準備
1. 新規のスプレッドシートを作成します。
2. シート名を「**設定**」に変更します。
3. **A列（1列目）**の1行目に見出し（例：「キーワード」）を書き、2行目以降に検索したいワードを1行ずつ入力します。
   - 例：`生成AI`, `DX`, `Google Cloud`, `サイバーセキュリティ`

### 2. GASの導入
1. スプレッドシートのメニュー [拡張機能] > [Apps Script] をクリックします。
2. 元のコードをすべて消去し、提供されたスクリプトを貼り付けます。
3. コード内の `SPREADSHEET_ID` を、作成したスプレッドシートのIDに書き換えます。
   - URL `https://docs.google.com/spreadsheets/d/◯◯◯/edit` の `◯◯◯` 部分がIDです。

### 3. Google Chat Webhookの設定
1. 通知したいGoogle Chatのスペースを開きます。
2. スペース名横の「▼」 > [アプリと統合] > [Webhookを追加] をクリックします。
3. 任意の名前（例：NewsBot）を付けて保存し、発行された **Webhook URL** をコピーします。
4. コード内の `WEBHOOK.URL` に貼り付けます。

## 設定（APP_CONFIG）の詳細解説

運用に合わせて `APP_CONFIG` オブジェクトの値を書き換えることで、挙動を最適化できます。

### 1. SEARCH（検索関連）
| 設定項目 | 内容 | 推奨値・例 |
| :--- | :--- | :--- |
| `PERIOD` | 検索対象とする期間 | `1d` (1日), `7d` (1週間), `1h` (1時間) |
| `MAX_RSS_ITEMS` | 1キーワードあたりの最大取得件数 | `10` 〜 `20` |
| `DISPLAY_LIMIT` | チャットに表示する最終的な記事数 | `15` 〜 `20` |

### 2. WEBHOOK（通知関連）
- `TITLE`: カードメッセージのヘッダーに表示されるタイトルです。

### 3. UI（デザイン関連）
- `TAG_COLOR`: キーワード（#タグ）の色。
- `DATE_COLOR`: 日付やスコアが表示される下部のテキスト色。

## カスタマイズ・設定方法

### A. 定期実行（トリガー設定）
毎日決まった時間に自動でニュースを配信するには、GASのトリガー機能を使います。
1. エディタ左側の **「アラーム時計アイコン（トリガー）」** をクリック。
2. 右下の **「+ トリガーを追加」** をクリック。
3. 実行する関数：`sendDailyItNewsToChat`
4. イベントのソース：「時間主導型」
5. 時間ベースのタイマー：「日付ベースのタイマー」を選択。
6. 時刻を選択（例：午前8時〜9時）して保存。

### B. 言語・地域の設定変更
現在は日本のニュース（日本語）を取得する設定ですが、海外のニュースを追いたい場合は `fetchAndScoreNews` 内の URL パラメータを変更します。
- **日本語 (日本)**: `&hl=ja&gl=JP&ceid=JP:ja`
- **英語 (米国)**: `&hl=en-US&gl=US&ceid=US:en`
- **英語 (英国)**: `&hl=en-GB&gl=GB&ceid=GB:en`

### C. スコアリングの仕組み
このスクリプトは、単に最新記事を出すのではなく、**「複数のキーワードにまたがってヒットした記事」**を重要とみなします。
- キーワード A, B, C すべてにヒットした記事 → `Score: 3` (最上位)
- キーワード A だけにヒットした記事 → `Score: 1`
これにより、ノイズの少ない「旬のトピック」を優先的にキャッチできます。

### D. スクリプトプロパティの活用（推奨）
Webhook URLなどをコードに直接書きたくない場合は、GASの「スクリプトプロパティ」を使用してください。
1. [プロジェクトの設定（歯車アイコン）] > [スクリプトプロパティ] に `WEBHOOK_URL` という名前でURLを保存。
2. コード内で `APP_CONFIG.WEBHOOK.URL` の代わりに `PropertiesService.getScriptProperties().getProperty('WEBHOOK_URL')` と記述します。

## トラブルシューティング
- **「権限がありません」と出る**: 初回実行時は、Googleアカウントへのアクセス許可が必要です。「詳細を表示」から「安全ではないページに移動」を選択し、許可をクリックしてください。
- **ニュースが1件も来ない**: スプレッドシートの「設定」シート名が正しいか、キーワードがA列の2行目以降に入っているか確認してください。
